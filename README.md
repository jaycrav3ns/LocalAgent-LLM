# LocalAgent-LLM
Front-end using an Ollama endpoint for local chat use.<br>Run Bash and Python.<br>Includes command chaining and Google search.<br>Working, but still very beta, may need some polish.

![screenshot](https://github.com/user-attachments/assets/825c70a0-38b1-4c7f-a58d-95d829e3dede)

- Uses a simple cookie for multi-user workspace isolation.<br>
- Model detection for your local Ollama instance.
- Change models by clicking the chevron next to the model name.<br>
- New chat, save memory, and download chat buttons.
